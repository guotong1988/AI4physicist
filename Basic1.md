
P(A|B) 用编程语言表示为` A = mnist_classify_model(B)`

如果当 B = 1 发生时 A = 1 发生的概率为 90%, （即模型在每个类的预测正确率为90%）

B 一共有5个的数据`[1,2,3,4,4]`，每个概率是`[0.2,0.2,0.2,0.4]`

则 
```
P(A = 1, B = 1) 
= P(A = 1|B = 1) * P(B = 1) 
= 0.2 * 0.9 = 0.18 
```
```
P(A = 4, B = 4) 
= P(A = 4|B = 4) * P(B = 4) 
= 0.4 * 0.9 = 0.36 
```
