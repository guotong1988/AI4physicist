# 《用编程语言解释数学》

P(A) 用python语言解释为：
```python
def function_P(generated_random_int_1to100):
    if generated_random_int_1to100 > 40: # A事件发生
        return 0.6
```
P(A) 用中文解释为：

事件A发生的概率
